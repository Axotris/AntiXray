# AntiXray
This is a Minecraft 1.21.11 Datapack which provides AntiXray. It is in early development and only supports diamonds in the overworld.

# ISSUES
- in stuctures like mineshafts, fake ores get generated at visible spots, because the check if a position is visible happens before structure generation
    -> possible solution is changing the fake ore generation from feature based to structure based
